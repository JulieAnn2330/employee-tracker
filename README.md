<h3 id='description'>Description of Project</h3>
<p>This project allows the user to track, add, delete and update employee and company information through the use of a dropdown menu of available functions and prompts that leads the user through the information needed. This project was a homework assignment for the University of Kansas BootCamp. This app stores information crucial to a business in one spot and allows the user a central location for updating and viewing information. I learned a lot about MySQL Workbench and node in developing this project. Particularly combining data from various tables. My project stands out because my tables are clean and easy to read. I updated the naming conventions in my sql databases and adjusted them so that they are easier to read. My project features a dropdown list of available options for the user and clearly defined prompts for the user to follow to input information.</p><br>

<h3 id='use'>How to Use the Project</h3>
<p>To use the project open the Employee-Tracker folder. In terminal run node server.js From the dropdown choose the task you want to complete, then follow the prompts for entering the information. A table will display after every task is complete and show up-to-date information for the user.</p><br>
